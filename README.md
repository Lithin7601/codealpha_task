# Basic Calculator in C Using Switch Case

## 📌 Project Overview

This project is a simple **Basic Calculator** developed in the **C programming language** using the **switch-case** statement. It performs basic arithmetic operations based on the user's choice.

## ✨ Features

* Addition (+)
* Subtraction (-)
* Multiplication (*)
* Division (/)
* User-friendly menu
* Input validation for division by zero

## 🛠️ Technologies Used

* C Programming Language
* GCC Compiler (or any C compiler)

## 📂 Project Structure

```
Calculator/
│── calculator.c
└── README.md
```

## 🚀 How to Compile and Run

### Compile

```bash
gcc calculator.c -o calculator
```

### Run

```bash
./calculator
```

**For Windows (MinGW):**

```bash
calculator.exe
```

## 📋 Sample Output

```
===== BASIC CALCULATOR =====

1. Addition
2. Subtraction
3. Multiplication
4. Division
5. Exit

Enter your choice: 1

Enter first number: 25
Enter second number: 15

Result = 40
```

## 📖 Concepts Used

* Switch Case
* Conditional Statements
* User Input (`scanf`)
* Arithmetic Operators
* Functions (optional)
* Loops (optional for repeated calculations)

## 🎯 Learning Objectives

* Understand the use of the `switch` statement.
* Practice arithmetic operations in C.
* Improve problem-solving skills using menu-driven programs.
* Learn basic input and output handling.
